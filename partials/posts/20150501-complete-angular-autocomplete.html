This is to introduce another autocomplete for INPUT or SELECT element. We can find many fully-featured autocomplet at github, but I was not fully satisfied with its implementation. The main reason of dissatisfaction is those does not applies to INPUT or SELECT tag.

Because autocomplete feature is a behaviour of a html element, I was expecting;

  • <input auto-complete source="...">
  • Or, <select auto-complete source="...">

I might have been lost in Google, but I could not find any AngularJS autocomplete module like the above. So, I made my own. I think it will not hurt you to have one more choice.

You can find source code here.

The usage is simple; just add auto-complete and source attribute to you INPUT or SELET tag like the following;

`<input auto-complete source="listOfChoices">`

or 

`<select auto-complete source="listOfChoices"></select>`

The source section could be an array, a url, or a function. The beauty of this is it does not require any more tags cucs as <ui-select-match>, <ui-select-choices>, or <autocomplete>. input or select tag itself is enough for autocomplete feature.

You can find more working examples;

  • INPUT tag autocomplete

  • SELECT tag autocomplete

  • SELECT tag multiple autocomplete

  • SELECT tag multiple autocomplete with custom css

  • Google geocode autocomplete - remote source

This module, angular-autocompete, requires only one mandatory attribute, source, and many optional ones. To know more about attributes, please visit github repository

I hope this module is helpful for someone.
